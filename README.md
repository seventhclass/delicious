================
小组合作经验总结
================

1. 任务划分需尽量降低耦合度，以尽量减少多人同时修改导致的冲突。
具体做法：
	每个人负责部分的结构（php）、样式（css）、行为（js）首先分开在单独的文件里编写，最后再整合到一起，集中处理冲突。
